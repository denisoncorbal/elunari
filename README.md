<!-- GETTING STARTED -->

## Getting Started

The system can be run locally using docker.

### Prerequisites

- Docker

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/denisoncorbal/elunari.git
   ```
2. Run the docker compose
   ```sh
   docker compose up -d --build
   ```
3. Access the homepage from the application on [localhost:4200](http://localhost:4200)
